# excel_automation_project02

## How to Run This Project

1. Install required libraries:
   pip install pandas openpyxl

2. Run the script:
   python automation.py

3. Output file:
   output_report.xlsx
## Project Workflow
1. Reads employee data from CSV file
2. Cleans missing values
3. Calculates department-wise average salary
4. Exports summary to Excel file

## Skills Demonstrated
- Python Automation
- Pandas for Data Analysis
- CSV to Excel Processing
- Real-world Office Automation
